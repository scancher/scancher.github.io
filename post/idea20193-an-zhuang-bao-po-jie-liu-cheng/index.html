IDEA2019.3安装包破解流程

2022-08-11

下载地址

链接：https://pan.baidu.com/s/19whCTqQ26HpKDgJLBTPCZg
提取码：ds3a

开始激活

安装IDEA结束后先点击试用进入idea主界面

配置破解补丁

进去过后，我们随便建个项目，点击菜单栏 Help -> Edit Custom VM Options:

注意：切记一定要通过 IDEA 来修改 .vmoptions 文件，不要手动直接去修改，现在 IDEA 针对反破解已经越来越严格了~

将下载的破解补丁文件jar包放在idea安装目录下防止后期误删，也可以放在其他目录，就是只要在上面配置中配置对有效路径就可以，建议存放在idea安装目录

注意：补丁全路径不要包含中文，否则破解失效

配置破解jar包的路径

在 .vmoptions 文件中配置我们补丁放置的全路径位置。注意替换成自己的存放路径

-javaagent:C:\Program Files\JetBrains\IntelliJ IDEA 2019.3\bin\jetbrains-agent.jar

重启idea填入激活码

配置好破解补丁以后，重启 IDEA.

重启成功以后，开始填入激活码，点击菜单栏 Help -> Register:

注意：下方的激活码必须配合上述的破解流程一起使用
